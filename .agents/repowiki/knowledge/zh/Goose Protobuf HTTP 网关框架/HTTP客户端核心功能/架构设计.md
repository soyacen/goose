- 采用函数选项模式（Functional Options）管理配置，通过 `option.go` 中的 `Options` 接口与 `NewOptions` 工厂方法实现灵活初始化。
- 定义 `Invoker` 与 `Middleware` 类型构建责任链模式，在 `middleware.go` 中通过递归辅助函数 `getInvoker` 实现中间件的嵌套调用。
- 集成 `protojson` 实现 Protobuf 消息与 HTTP 载荷的双向转换，`encoder.go` 与 `decoder.go` 分别处理请求序列化与响应反序列化。
- 内置 `resolver` 子包提供基于 Scheme 的 URL 解析扩展点，利用 `sync.Map` 注册机制支持 `http/https` 默认解析器及自定义解析逻辑。
