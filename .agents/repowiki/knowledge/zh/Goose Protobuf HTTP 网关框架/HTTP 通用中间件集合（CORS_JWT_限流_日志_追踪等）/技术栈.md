- JWT 认证基于 `github.com/golang-jwt/jwt/v5`，支持 HS512/RSA 等多种 `SigningMethod`。
- 可观测性通过 `go.opentelemetry.io/contrib/instrumentation/net/http/otelhttp` 注入 Span，并复用 `goose.ExtractRouteInfo` 填充 `http.route`/`rpc.method` 属性。
- 自适应限流 `limiter` 实现字节跳动 BBR 算法的纯内存滑动窗口统计（`rollingCounter` 桶数组 + `atomic.Pointer[time.Time]` 记录丢弃时间），不依赖 Redis 等外部存储。
